# Server-Controller
A macropad designed to control the status of a Raspberry Pi server without having to plug in a monitor. I has three buttons and a small screen to display status updates.

<img width="942" height="918" alt="Separated Case" src="https://github.com/user-attachments/assets/934ee840-ce72-45bf-afdf-ecbc406171f8" />

BOM: 
- 1 XIAO RP2040
- 1 SSD1306 OLED Display
- 3 Cherry MX Switches
- 3 Blank DSA Keycaps
- 4 M3x16 Bolts
- 4 M3 Heatsets
